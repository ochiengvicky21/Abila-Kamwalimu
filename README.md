# Abila Kamwalimu Restaurant Website

A modern, responsive restaurant website for **Abila Kamwalimu Restaurant** located in **Homa Bay, Kenya**.

Designed and developed by **Kijani Systems**.

---

# Overview

This project is a professional restaurant website built using pure HTML, CSS and JavaScript. It is lightweight, responsive, mobile-friendly and suitable for deployment on any static web hosting service.

The website showcases the restaurant's brand, menu, gallery, events and contact information while providing an attractive user experience.
